Q5: When I use Bootbox, the header's all messed up. What gives?

A: Bootbox 6 is intended to work with any currently-supported version of Bootstrap — meaning you can use it with Bootstrap 3.4.1 and newer. If you're seeing something similar to this… 

bootbox.alert({ 
    title: 'My title',
    message: "Your message here…" 
});

… then you're probably using Bootstrap 4 or newer and Bootbox 4. The markup structure and CSS rules for Bootstrap 4+ are not compatible with the markup used in Bootstrap 3 modals, which is what Bootbox 4 generates. At the moment, there are no plans to update Bootbox 4, so please upgrade to Bootbox 5 or newer if you are able.
